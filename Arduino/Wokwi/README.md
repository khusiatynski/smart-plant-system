# Arduino Wokwi

Pliki do symulacji projektu w Wokwi.

- `sketch.ino` - kod demonstracyjny dla symulacji
- `diagram.json` - układ elementów i połączeń

Link do projektu online:

`https://wokwi.com/projects/461119979508150273`

Założenia:

- BME280 po I2C pod adresem `0x76`
- wilgotność gleby na `A0`
- światło na `A1`
- poziom wody na `A2`
- LED grow na `D6`
- przekaźnik pompy na `D7`
