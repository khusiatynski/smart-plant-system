# Schemat

Folder na schematy połączeń, zrzuty z programu CAD oraz zdjęcia okablowania.

## Proponowany układ połączeń

- `Arduino Nano`
- `BME280` po I2C:
  - `VCC -> 5V`
  - `GND -> GND`
  - `SDA -> A4`
  - `SCL -> A5`
- `Czujnik wilgotności gleby -> A0`
- `Czujnik światła / LDR -> A1`
- `Czujnik poziomu wody -> A2`
- `Przekaźnik pompy -> D7`
- `Sterowanie LED grow -> D6`

## Co warto tu dodać

- schemat blokowy układu
- pełny schemat połączeń
- zdjęcia montażu
- opis zasilania i zabezpieczeń
