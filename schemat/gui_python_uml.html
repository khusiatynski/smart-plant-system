🎨 Wzorzec Strategy Pattern

Interfejs: DeviceConnection

Implementacje:

  • MockDeviceConnection: Symulacja danych dla trybu demo (bez sprzętu)
  • UartDeviceConnection: Komunikacja UART z prawdziwym Arduino

Zalety:

  • Łatwe przełączanie między trybem demo a rzeczywistym sprzętem
  • Testowanie GUI bez Arduino
  • Fallback do MockDevice gdy UART się rozłączy

🔄 Przepływ danych

  1. PlantApp._tick(): Cykl 2s (timer tkinter):
    • Wywołuje service.refresh_data()
    • Service odczytuje dane z device_connection.read_plant_data()
    • Service stosuje automatykę (jeśli włączona)
    • Service loguje dane przez data_logger.log()
    • Service aktualizuje history (max 40 próbek)
  2. PlantApp.refresh_view(): Aktualizacja UI:
    • Odświeża 3 zegary BME (temp, humidity, pressure)
    • Aktualizuje karty metryki (soil, light, water)
    • Rysuje wykresy trendów (soil, light)
    • Ustawia stan przełączników (AUTO/RĘKA, pompa, LED)
  3. Akcje użytkownika:
    • Przełączniki → service.set_pump_manual(), service.set_led_manual()
    • Konfiguracja progów → service.update_thresholds()
    • Połączenie UART → service.connect_to_uart()
    • Tryb demo → service.switch_to_demo_mode()

📊 Komponenty UI

Okrągłe zegary (GaugeCard):

  • Temperatura: 0-40°C (kolor #f5b041)
  • Wilgotność powietrza: 0-100% (kolor #5dade2)
  • Ciśnienie: 970-1045 hPa (kolor #f39c12)

Karty metryki (MetricCard):

  • Wilgotność gleby: 0-100% z paskiem postępu
  • Natężenie światła: 0-100% z paskiem postępu
  • Poziom wody: 0-100% z paskiem postępu

Wykresy trendów (TrendChart):

  • Historia 40 ostatnich pomiarów
  • Gładkie krzywe (smooth=True)
  • Siatka pomocnicza (5 poziomów 0-100%)

Sterowanie (ModeControlCard):

  • Przełącznik AUTO/RĘKA (zielony/czerwony)
  • Przełącznik pompy ON/OFF (zielony/czerwony)
  • Przełącznik LED ON/OFF (zielony/czerwony)

💾 Logowanie danych (CSV)

Ścieżka: logs/plant-log.csv

Nagłówek:

Częstotliwość: Jeden wiersz co 2 sekundy (podczas pracy aplikacji)

🎯 Automatyka (PlantControllerService)

Algorytm z histerezą:

# Pompa
if soil_moisture < threshold.min_soil_moisture and water_level > 5.0:
    pump_on = True
elif soil_moisture >= threshold.max_soil_moisture:
    pump_on = False

# LED Grow
if light_level < threshold.min_light_level:
    led_on = True
elif light_level >= threshold.max_light_level:
    led_on = False

# Zabezpieczenie pompy (nadrzędne)
if water_level <= 2.0:
    pump_on = False  # ZAWSZE wyłącza pompę

Domyślne progi:

  • Wilgotność gleby: 40-60%
  • Natężenie światła: 35-65%
  • Próg wody (bezpieczeństwo): 2%
